基於符號執行的自動化C語言程式漏洞檢測工具

Automated Vulnerability Detection Tool for C Language using Symbolic Execution

指導教授： 江季瀚

成員： 蘇靜山 楊博宇 王顥霖 謝彥宭

簡介：

  本研究致力於開發一個自動化漏洞檢測工具，以解決軟體開發中的設計瑕疵和邏輯錯誤問題。將利用符號執行技術和開源程式分析框架 angr，主要針對常見的漏洞進行檢測，包括堆疊溢出、格式化字串漏洞、釋放後重用、雙重釋放和堆積溢位。
  本系統以 C 語言為目標，專注於可執行和可鏈結格式的分析。透過本研究，希望提供一個快速準確的自動化漏洞檢測工具，提升軟體系統的安全性。

使用說明：

  本網站主要分為檔案分析和分析結果列表兩個部分。 在檔案分析部分，使用者可上傳已編譯的執行檔，選擇欲進行的模組，系統將上傳的檔案傳送至伺服器進行分析，分析完成後將顯示在分析結果列表中。 分析結果列表中顯示以往的分析記錄，使用者可點擊「查看」以檢視該次分析的結果。
